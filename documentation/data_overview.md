Dataset Description
There are 13 variables in the dataset. The variable drug_category is the target/ response variable.

File descriptions
data_minihackathon_train - the training set
data_minihackathon_test - the test set without the target/response variable
sampleSubmission.csv - a sample submission file in the correct format. Note that column headings should be ID and drug_category
Attribute_Information.pdf - Variable coding and description
Train your models to predict "drug_category" using "data_minihackathon_train" dataset, obtain predictions for the "data_minihackathon_test" dataset from your model, save the predictions in a .csv file in the given format and upload using the "Submit Prediction" button.

Data Fields
ID: ID is number of record in original database
Age: Age is the age of participant and has one of the values
Gender: Gender is gender of participant
Education: Education is level of education of participant and has one of the values
Country: Country is country of current residence of participant and has one of the values
Ethnicity: Ethnicity is ethnicity of participant and has one of the values
Nscore: Nscore is NEO-FFI-R Neuroticism.
EScore: Escore (Real) is NEO-FFI-R Extraversion
Oscore: Oscore (Real) is NEO-FFI-R Openness to experience
Ascore: Ascore(Real) is NEO-FFI-R Agreeableness
Cscore: Cscore (Real) is NEO-FFI-R Conscientiousness
Impulsive: Impulsive (Real) is impulsiveness measured by BIS-11
Sensation: SS(Real) is sensation seeing measured by ImpSS
